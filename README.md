# Querystring usage

> The small example how use the lib


## Install

```
npm i
```

## Convert to AMD

```
npm run amd
```

## Example

```
npm start
```

Click on `Try the link` or
try input another query string: `http://127.0.0.1:3333/?a=1&b=2`
